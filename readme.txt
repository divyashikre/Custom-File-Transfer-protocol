Instructions to execute the server and client code:

Source Files : server.c and client.c
Script Files : script_server.sh and script_client.sh

1. Initially type make to compile the server.c and client.c
2. Make script_server.sh and script_client.sh as executable
3. In server run ./script_server.sh <client ip address> output3
4. In client run ./script_client.sh <server ip address> <file name>
